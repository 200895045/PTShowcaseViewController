How To Get It
=============

```bash
$ git clone --recursive https://github.com/exalted/PTShowcaseViewController.git PTShowcaseViewController-exalted
$ open PTShowcaseViewController-exalted/Examples/ShowcaseDemo/ShowcaseDemo.xcodeproj
```
